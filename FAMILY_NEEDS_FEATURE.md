# ميزة إدارة احتياجات العائلات

## نظرة عامة
تم إنشاء نظام كامل لإدارة احتياجات كل عائلة بشكل منفصل مع تتبع المبالغ والمستخدمين.

## الميزات
- جدول وسيط بين العائلات والاحتياجات
- تخزين المبلغ التقديري لكل احتياج
- تتبع المستخدم الذي أضاف/عدّل
- تتبع وقت الإنشاء والتعديل
- حالات: معلق، تم التنفيذ، ملغي

## API Endpoints
1. GET /api/families/{family_id}/needs - احتياجات العائلة
2. POST /api/families/{family_id}/needs - إضافة احتياج
3. PUT /api/families/{family_id}/needs/{id} - تحديث احتياج
4. DELETE /api/families/{family_id}/needs/{id} - حذف احتياج
